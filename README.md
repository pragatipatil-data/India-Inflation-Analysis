# 📊 India Inflation vs Common Man (2015-2024)

## Overview
Analysis of India's inflation impact on common man's
purchasing power using World Bank and RBI data.

## Tools Used
- PostgreSQL — Data analysis and SQL queries
- Excel — Pivot Tables and trend analysis
- Power BI — 3 page interactive dashboard

## Key Findings
- Rs 100 in 2015 = Rs 59 in 2024 (40% loss!)
- Food inflation highest impact on common man
- COVID 2020 caused spike to 6.62%
- Fuel inflation peaked at 21.7% in 2021

## Data Sources
- World Bank — CPI Inflation Data
- RBI — Category wise inflation
